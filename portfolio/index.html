Portfolio

Flyover of Switzerland

Using height map data from https://www.swisstopo.admin.ch (with noise added to reduce artefacts) as a displacement map, the scene was rendered in Blender. A procedurally generated texture was used to color the heightmap.

Waveform Collapse

Implemented the WFC algorithm in Go and Python. The resulting scenes are rendered in Blender. Here's two examples of an array of scenes using two tilesets.

This was joint work with a friend.

Part of this code is open-sourced: the parser for magicavox .vox files.

Go Game

Goban (board) and pieces (slate and clamshell), modelled and rendered in Blender. Slate, clamshell and wood materials are procedural PBR shaders, built entirely in Blender. The game is from the AlphaGo-Sedol match in 2016.
